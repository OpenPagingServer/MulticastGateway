# MulticastGateway
Allows easy transmission of multicast packets between WAN endpoints and in VPN tunnels

Multicast packets are sent as Unicast and resent as Multicast by the gateway.

## Supported configugrationsn
The OPS multicast gateway supports:
- Open Paging Server (v0.4.0 or later) <---> Multicast Gateway (Native support)
- Multicast Gateway <---> Multicast Gateway (for any multicast traffic generated by any network device)
